# VARPAD
VARPAD - text editor with the feature of creating variables and using them right in the text

## Controls
 F10 - save your text file;
      Shift + Scroll - scroll horizontally

## VARIABLE CREATION
 In the var section (||VAR:  ...  :END||) you can interact with variables:
 - $name = value   (Creates a variable)
 - %name = value   (Changes existing variable's value)

### VARIABLE TYPES
 - Integer 123
 - Text "abc"

### OPERATIONS
 - Multiply *
 - Add +
 - Subtract -
 - Divide /
